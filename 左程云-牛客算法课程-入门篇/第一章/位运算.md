## 异或运算 ^

1100 ^ 1010  = 0110  
异或运算： 相同为0，不同为1  
更好的理解：【不进位相加】

### 性质

a ^ a = 0
0 ^ b = b  
a ^ a ^ b = b

交换律 a^b=b^a  
结合律 (a^b)^c=a^(b^c) 
故 只要是同一批数异或，结果一定一样。

#### a交换b
```
int a = 甲
int b = 乙

a= a^b  // a=甲^乙，b=乙
b= a^b  // a=甲^乙, b=甲^乙^乙=甲^0=甲
a= a^b  // a=甲^乙^甲=乙，b=甲
```
无需变量temp 即可交换a 和 b ， 

a与b的值可以相等，但a和b不能指向一块内存。
同样一块内存跟自己异或，会被置为0；
 如 arr[i]， arr[j]  i必须!=j  否则 arr[i] 被洗成0；
